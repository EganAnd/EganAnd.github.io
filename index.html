Projects

Project 1: Machine Learning: Predicting Employee Retention using HR Data

This project is the capstone project from the Google Advanced Data Analytics Course. The goal was to analyze human resource data to build a predictive model for employee retention. The project involves extensive exploratory data analysis to understand the data, followed by building and fine-tuning machine learning models. The key libraries used include pandas, matplotlib, seaborn, scikit-learn, and XGBoost. The notebook contains detailed visualizations and explanations to help understand the model's performance.
